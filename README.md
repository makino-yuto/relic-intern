# Snap Diary

写真と一緒に日記を記録できる Web アプリです。

自分自身が日記を書くために開発しました。
「日記を書きたいけど続かない」という課題を、できるだけシンプルに解決することを目指しています。

本アプリは、「株式会社Relic 2028年卒向け就活イベント」にて制作したプロトタイプであり、将来的に Android ネイティブアプリとしてリリースすることを前提に設計・開発を行いました。


アンドロイドアプリのリポジトリ:
[diary_app](https://github.com/makino-yuto/diary_app)


## Features

* 写真付き日記投稿
* カレンダー表示
* ログイン機能
* クラウド保存

## Tech Stack

* React
* TypeScript
* Vite
* Supabase
* Vercel
